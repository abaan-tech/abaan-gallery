Convert WP Gallery to Fancybox Thumbnail Gallery.

== Description ==

> Abaan WP Galley gives WordPress ures the ability to show default wp gallery like fancybox thumbnail gallery. The plugin also have the ability to show youtube video in fancybox by clicking on thumbnail.

For complete details visit [baaan.info](http://baaan.info/abaan-gallery/).  

== Installation ==

1. Upload `abaan-gallery` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

The Duplicator requires php 5.3 or higher.
